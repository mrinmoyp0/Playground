# Offline Chess Duel

A local two-player chess app built with Python and Streamlit.

## Run it

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Included

- Offline local play for two people on one device
- Legal move validation
- Castling, en passant, and pawn promotion
- Check, checkmate, and stalemate detection
- Move history, captured pieces, undo, and new game controls
